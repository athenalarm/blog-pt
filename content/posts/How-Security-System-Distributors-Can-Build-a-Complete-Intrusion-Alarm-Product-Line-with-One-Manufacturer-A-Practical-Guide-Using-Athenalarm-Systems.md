As configurações básicas suportam 16 zonas com fio mais 30 zonas sem fio, expansíveis para 1.656 zonas de barramento usando módulos endereçáveis — uma escalabilidade incomparável para projetos comerciais e de vários locais.

- AS-9000FF (PSTN + 4G + TCP/IP)
